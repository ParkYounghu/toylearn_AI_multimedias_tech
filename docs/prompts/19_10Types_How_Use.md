가상 문제를 준 후, 미비점이 있는 지 확인하고 그에 맞는 보완점 체크

2. 예시 기반

3. 역할 부여

4. 제약 조건

5. 질문-응답

6. 체인오브 스루트

7. 페르소나

## 가상 문제 상황

페르소나: 광고 대행사의 크리에이티브 디렉터 (Creative Director)
배경: 당신은 최근 클라이언트로 합류한 'AI 기반 업무 자동화 SaaS' 스타트업의 신규 캠페인 런칭을 총괄하게 되었습니다. 클라이언트로부터 20페이지 분량의 상세한 **기술 명세서(Technical Specification)**와 기능 리스트를 전달받았습니다.
핵심 문제: 주 타겟 고객은 MZ세대 직장인입니다. 하지만 이 제품은 기능이 복잡하고 기술 용어가 많아, 기존의 텍스트 중심 설명 자료가 "너무 지루하고 복잡하다"라는 내부 테스트 피드백을 받았습니다. 제품의 복잡한 기능을 직관적이고 '힙하게' 전달할 새로운 멀티미디어 접근 방식이 절실한 상황입니다.
해결 과제: 클라이언트가 제공한 20페이지 분량의 기술 명세서를 기반으로, 타겟 고객의 관심을 즉각적으로 사로잡을 수 있는 30초 분량의 '인스타그램 릴스' 또는 '틱톡' 숏폼 영상 광고 시나리오 3가지를 구체적으로 도출해야 합니다. (각 시나리오는 핵심 카피, 비주얼 컨셉, 활용할 BGM 아이디어를 포함해야 합니다.)
2. 훈련 초점 (힌트)
이 문제는 [창의적 아이디어 생성] 및 [텍스트 변환(Text Transformation)] 능력을 평가 (복잡한 기술 문서를 매력적인 멀티미디어 시나리오로 변환해야 합니다.)

## 문제 해결 프롬프트 
```
너는 광고 대행사의 **베테랑 크리에이티브 디렉터**이며, 팀을 이끄는 매니저(나)에게 **최종 기획안을 보고하는 형식**으로 답변해야 한다. 답변은 **창의적이고 실행 가능한 실무적 톤**을 유지해야 한다.

**[상황 배경 및 조건부 생성]**
클라이언트 'A'(AI 기반 업무 자동화 SaaS)의 신제품 런칭 캠페인을 총괄한다.
1.  클라이언트로부터 **20페이지 분량의 상세한 기술 명세서**를 전달받았으며, 이 명세서가 유일한 콘텐츠 소스다. (제약 조건)
2.  **주 타겟은 MZ세대 직장인**이다.
3.  기존 설명 자료가 **"너무 지루하고 복잡하다"**는 내부 피드백을 받았다. (핵심 문제)

**[해결 과제 및 지시 사항]**
당신은 기술 명세서의 복잡한 기능을 직관적이고 '힙하게' 전달할 **30초 분량의 숏폼 영상 광고 시나리오 3가지**를 구체적으로 도출해야 한다.

**1. 분석 및 창의적 목표 (CoT 및 인스턴스 기반 필수):**
* **Chain of Thought:** '기술 명세서'의 복잡성을 **'MZ세대 직장인의 해방/편의성'**이라는 가치로 변환하는 과정을 명확히 할 것.
* **인스턴스 기반:** 시나리오 구성 시, 당신의 **5년 이상 경력 데이터**를 활용하여 **최근 1년 간 성공적인 MZ세대 대상 업무 효율 SaaS 광고**의 **크리에이티브 트렌드**를 참고할 것.

**2. 출력 형식 및 통제 (역할 부여, 지시형, 예시 기반):**
답변은 반드시 하나의 **markdown 코드블럭**을 사용하며, 아래 형식을 준수해야 한다. 시나리오 3가지는 각각 제목을 붙여 구분할 것.

| 항목 | 기획 내용 | 비고 (기술 명세서 활용) |
| :--- | :--- | :--- |
| **광고 컨셉** | (시나리오 전체의 주제) | (기술 명세서의 어떤 기능/기술을 활용했는지 명시) |
| **핵심 카피** | Hook: (3초 이내 후킹 메시지) / Benefit: ... / CTA: ... | (광고 메시지 전략 설명) |
| **비주얼 컨셉** | (장면, 조명, 구도 등 상세 설명) | (비주얼의 의도 설명) |
| **BGM 아이디어** | (예시 포함, MZ 트렌드 댄스곡 또는 Lo-fi) | (ex. MZ 트렌드 댄스곡 (ex. 르세라핌 'EASY' 비트) 형태로 제시할 것) |
```
## 프롬프트 실행 결과

## 💡 AI 기반 SaaS 런칭 캠페인: 30초 숏폼 영상 광고 시나리오 최종 기획안

**보고자:** 크리에이티브 디렉터
**타겟:** MZ세대 직장인 (인스타그램 릴스/틱톡)
**목표:** 20페이지 기술 명세서를 **'힙하고 직관적인 가치'**로 변환하여 제품 인지 및 유입 유도.

| 항목 | 기획 내용 | 비고 (기술 명세서 활용) |
| :--- | :--- | :--- |
